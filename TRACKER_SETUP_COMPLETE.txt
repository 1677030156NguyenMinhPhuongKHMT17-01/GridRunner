================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: "ModuleNotFoundError: No module named 'mediapipe'"
SOLUTION: pip install mediapipe==0.10.13

ISSUE: Camera not found / black screen
SOLUTION: 
  1. Close Zoom, Teams, other apps using camera
  2. Check camera in Settings > Devices > Camera
  3. Try different camera: python hand_tracker_server.py 1
  4. Restart computer

ISSUE: Hands not detected / "No hands - Show both hands!"
SOLUTION:
  1. Ensure adequate lighting
  2. Show both hands clearly
  3. Lower detection confidence: edit min_detection_confidence=0.3

ISSUE: Tracking jittery / laggy
SOLUTION: Adjust Kalman parameters in hand_tracker_server.py
  - Lower process_var for smoother (less responsive)
  - Lower measurement_var for more responsive (more jittery)

ISSUE: Game doesn't receive hand data
SOLUTION:
  1. Verify tracker runs in GUI mode first
  2. Check UDP port 5555 is not blocked (firewall)
  3. Verify main.gd has "hand_tracker_server.py" in res://
